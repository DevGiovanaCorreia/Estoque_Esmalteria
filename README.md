#  Sistema de Controle de Estoque - Esmalteria

Este projeto é um sistema desktop desenvolvido em Java com o objetivo de controlar o estoque de uma esmalteria, permitindo o gerenciamento de produtos e o registro de movimentações de entrada e saída.

---

## 📌 Sobre o projeto

O sistema foi criado para facilitar o controle de itens em estoque, evitando perdas e mantendo um histórico das movimentações realizadas.

A aplicação permite cadastrar produtos, editar informações, excluir registros e controlar automaticamente a quantidade disponível em estoque com base nas entradas e saídas realizadas.

---

## ⚙️ Funcionalidades

- Cadastro de produtos  
- Edição e atualização de dados  
- Exclusão de produtos  
- Listagem em tabela  
- Controle automático de estoque  
- Registro de entrada de produtos  
- Registro de saída de produtos  
- Histórico de movimentações  

---

## 🧱 Tecnologias utilizadas

- Java (Swing)
- JPA (Jakarta Persistence)
- Hibernate
- MySQL
- Maven

---

## 🔄 Funcionamento

O sistema funciona a partir de um fluxo simples:

1. O usuário cadastra os produtos  
2. Os produtos são exibidos em uma tabela no menu principal  
3. Ao realizar uma movimentação, o usuário informa se é entrada ou saída  
4. O sistema atualiza automaticamente o estoque  
5. Todas as movimentações são registradas no banco de dados  

---

## 📚 Objetivo

Este projeto foi desenvolvido com foco em aprendizado e prática de conceitos importantes como:

- CRUD com banco de dados  
- Persistência com JPA  
- Organização em camadas (DAO, entidades e interface)  
- Manipulação de dados em aplicações reais
